
# Galaxy Store

## Project Description

An online clothing store to buy clothes.

## Technologies Used

* Angular
* Spring Boot
* Spring Web
* Spring Data JPA

## Features

List of features ready and TODOs for future development
* Signup
* Login
* View Products
* Search Products
* Add to Cart

## Getting Started

* Clone the project
* Import the project into STS
* Provide Maven Update for the project in STS
